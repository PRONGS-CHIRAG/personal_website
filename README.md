# Chirag N Vijay — Premium AI Founder Portfolio

This repository contains the implementation brief and build plan for a **modern, premium, fully responsive personal portfolio website** using:

- **Next.js 14+** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**

The website should represent an **AI Engineer & Entrepreneur** profile with a polished, high-end, startup-native visual identity (not a generic student portfolio).
